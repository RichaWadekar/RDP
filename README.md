# Playwright Automation Skeleton

This is a basic Playwright project skeleton for end-to-end testing.

[![Allure Report](https://github.com/RichaWadekar/RDP/actions/workflows/publish-allure-report.yml/badge.svg)](https://RichaWadekar.github.io/RDP/)

Public report: https://RichaWadekar.github.io/RDP/

## Getting Started

1. Install dependencies:
   ```
npm install
   ```
2. Run tests:
   ```
npm test
   ```

## Project Structure
- `tests/` - Example Playwright tests
- `playwright.config.js` - Playwright configuration
- `package.json` - Project metadata and scripts
